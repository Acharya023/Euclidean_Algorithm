# Euclidean_Algorithm
The x and y are the result for inputs a and b |
a.x + b.y = gcd |
And x1 and y1 are results for inputs b%a and a |
(b%a).x1 + a.y1 = gcd |
We put b%a = (b- (b//a).a) |
(b - ((b//a).a).x1 + a.y1 = gcd |
Hence,    b.x1 + a.(y1 - (b//a).x1) = gcd |
After comparing |
x = y1 - (b//a)*x1 |
y = x1 |
